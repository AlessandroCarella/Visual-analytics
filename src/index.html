Additional Information

There is a black thin line that links the entities under investigation between each other.
This connection is not present in the original dataset but is included to highlight potentially significant relationships between these entities.

The nodes in yellow are both left and right clickable. A left click on them will expand that particular node or entity as both source and target and will respectively show its targets and sources (this depends on the entities linked to that node in the first place).
The right click opens a custom context menu that shows the 3 possible options:
- Expand as source
- Expand as target
- Expand as both
